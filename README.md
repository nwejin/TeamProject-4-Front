**웨더러블은*은 오프라인의 개인 옷장을 온라인으로 옮겨 나만의 옷장을 손안에서 관리할 수 있는 웹 플랫폼입니다.

옷장 페이지에서 직접 사진과 옷 정보를 입력하여 옷을 등록할 수 있으며, 쇼핑몰 사이트에서 크롤링해온 데이터에서 옷 정보를 검색하여 등록이 가능합니다.

open AI를 활용하여 내 옷장 속 옷 데이터와 현재 날씨 정보에 맞는 코디를 추천해주며, 

나만의 코디를 직접 등록이 가능합니다.
